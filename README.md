# UrbanShield

UrbanShield is a hackathon base project for emergency resource allocation planning in Bangalore.

This repository currently contains only the clean starter structure: Vite, React, Tailwind CSS, Recharts, sample data, and a mock API service contract.

## Setup

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Project Structure

```text
src/
  components/
  data/
  services/
  utils/
  types/
  App.jsx
  main.jsx
```

## Branch Workflow

- `main` -> base project structure
- `frontend-ui` -> UI work
- `ai-integration` -> AI logic and allocation service integration
